# AE_final_project

### 介绍
自动化综合设计
模拟现实环境的1:10无人驾驶小车项目
#### 小车具体

### 计划
#### 第一周
1. yolo 移植树莓派
2. 场地设计与搭建
3. 车模购买，组装，实现
4. 确定小车需要实现功能
5. 进行人员分组

###组别分类
####干活组：
识别算法，小车控制算法，硬件设计  
####后勤组：
场地搭建，搬运实验器材，拿取快递，实验数据处理（例：数据集抠图）  
####财务组：
实验器材购买，消费清单以及发票整理  
####报告组：
全程跟随实验进行记录，最终整理班级报告
